## AUDIO SYSTEM DESIGN — MetaSounds Blueprint

### Adaptive Music Layers
```
Layer 0 — BASE: Percussion (bone/wood) + low drone — always playing
Layer 1 — EXPLORATION: Bird calls, wind, distant water — when no threat
Layer 2 — ALERT: Rhythmic percussion intensifies — predator detected nearby
Layer 3 — COMBAT: Full percussion + dissonant tones — active attack
Layer 4 — DEATH: Silence, single low tone fade
```

### Trigger Conditions (Blueprint Logic)
```
ThreatLevel 0.0-0.2 → Layer 0+1 (peaceful exploration)
ThreatLevel 0.2-0.5 → Layer 0+2 (alert state)
ThreatLevel 0.5-0.8 → Layer 0+3 (combat)
ThreatLevel 0.8-1.0 → Layer 0+3+4 (near death)
```

### Dinosaur Audio Signatures
- **T-Rex**: Sub-bass rumble (40-80Hz) + ground shake LFE at 60m radius
- **Raptor Pack**: High-pitched chirps + coordinated call-response pattern
- **Brachiosaurus**: Deep resonant call + heavy footfall at 100m radius
- **Pteranodon**: Wing beats + high screech on dive attack
