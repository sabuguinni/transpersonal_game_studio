# TIME-OF-DAY LIGHTING SYSTEM - BLUEPRINT SPECIFICATION
# Agent #8 - Lighting & Atmosphere Agent
# Cycle: PROD_CYCLE_AUTO_20260617_001

## SYSTEM OVERVIEW

Dynamic day/night cycle for prehistoric survival game with realistic atmospheric transitions.
Designed for Blueprint implementation (no C++ required per global directive).

## CORE PARAMETERS

### Time Progression
- **Day Length**: 24 real minutes = 1 game day (60x time compression)
- **Time of Day**: Float 0.0-24.0 (0=midnight, 6=dawn, 12=noon, 18=dusk, 24=midnight)
- **Time Speed Multiplier**: 60.0 (adjustable for testing/gameplay)

### Key Lighting Moments
1. **Dawn** (5:00-7:00): Warm orange/pink horizon, low intensity
2. **Morning** (7:00-10:00): Rising sun, increasing intensity, golden tint
3. **Noon** (10:00-14:00): Peak intensity, neutral white light, minimal shadows
4. **Afternoon** (14:00-17:00): Descending sun, warming color temperature
5. **Dusk** (17:00-19:00): Deep orange/red horizon, long shadows, dramatic lighting
6. **Twilight** (19:00-21:00): Fading light, purple/blue sky, stars appearing
7. **Night** (21:00-5:00): Moonlight only, cool blue tint, low visibility

## BLUEPRINT IMPLEMENTATION GUIDE

### Required Actors
1. **BP_TimeOfDayManager** (Blueprint Actor)
   - Tick enabled
   - References: DirectionalLight, SkyAtmosphere, SkyLight, ExponentialHeightFog
   - Variables: CurrentTime (float), TimeSpeed (float), bPaused (bool)

2. **Event Graph Logic**
   ```
   Event Tick
   ├─ If NOT Paused
   │  ├─ CurrentTime += DeltaTime * TimeSpeed / 3600.0
   │  ├─ If CurrentTime > 24.0: CurrentTime -= 24.0
   │  └─ Call UpdateLighting(CurrentTime)
   └─ UpdateLighting Function
      ├─ Set Directional Light Rotation (curve lookup)
      ├─ Set Directional Light Intensity (curve lookup)
      ├─ Set Directional Light Temperature (curve lookup)
      ├─ Set Sky Atmosphere Parameters (curve lookup)
      ├─ Set Volumetric Fog Density (curve lookup)
      ├─ Set Volumetric Fog Color (curve lookup)
      └─ Set Skylight Intensity (curve lookup)
   ```

### Curve Assets Required
- **Curve_SunRotationPitch** (Float Curve, 0-24 hours)
- **Curve_SunRotationYaw** (Float Curve, 0-24 hours)
- **Curve_SunIntensity** (Float Curve, 0-24 hours)
- **Curve_SunTemperature** (Float Curve, 0-24 hours)
- **Curve_FogDensity** (Float Curve, 0-24 hours)
- **Curve_SkylightIntensity** (Float Curve, 0-24 hours)

### Exposed Parameters (Blueprint Editable)
- **Time Speed Multiplier** (default 60.0, range 1.0-1000.0)
- **Starting Time** (default 8.0 = 8:00 AM)
- **Enable Day/Night Cycle** (bool, default true)
- **Override Time** (float, for testing specific lighting conditions)

## PERFORMANCE OPTIMIZATION

### Update Frequency
- Lighting parameters update every tick (smooth transitions)
- Curve lookups cached per frame (no redundant calculations)
- Sky atmosphere updates limited to 0.1-second intervals (imperceptible)

### LOD Considerations
- Volumetric fog distance scales with time of day (reduced at night)
- Shadow cascade distances increase during golden hour (long shadows)
- Skylight recapture interval: 5 minutes game time (performance vs quality)

## GAMEPLAY INTEGRATION

### Survival Mechanics
- **Temperature**: Colder at night (affects player stamina)
- **Visibility**: Reduced at dawn/dusk/night (predator advantage)
- **NPC Behavior**: Diurnal/nocturnal species activity cycles
- **Quest Triggers**: Time-specific events (e.g., "Meet at dawn")

### Cinematic Moments
- **Golden Hour Quests**: Dramatic lighting for key story beats
- **Night Stealth**: Low visibility enables infiltration gameplay
- **Storm Integration**: Time-of-day + weather system combination

## ARTISTIC INTENT

The time-of-day system serves:
1. **Realism**: Authentic prehistoric atmosphere with scientifically plausible lighting
2. **Gameplay Variety**: Different challenges/opportunities at different times
3. **Emotional Pacing**: Dawn = hope, noon = clarity, dusk = danger, night = fear
4. **Visual Spectacle**: Golden hour and twilight as signature cinematic moments
