## ARTISTIC INTENT

The volumetric fog system creates a prehistoric atmosphere that:
1. **Enhances depth perception** - Layered atmospheric haze adds scale
2. **Supports cinematic lighting** - God rays and sun shafts through mist
3. **Varies by location** - Denser fog in valleys/rivers, lighter in highlands
4. **Realistic time-of-day** - Fog density and color shift with sun angle
5. **Immersive environment** - Warm, primordial atmosphere distinct from modern landscapes

## TECHNICAL NOTES

- All fog properties set via Python Remote Control API
- Compatible with UE5.5 Lumen rendering pipeline
- Fog zones implemented as scaled point light actors (label-based system)
- Map auto-saved after fog configuration
- No C++ compilation required - runtime configuration only
