## LOD Audit Results

BasicShape placeholders (Cylinder, Sphere, Cube) used for resource nodes have **1 LOD only**.
This is acceptable for the current prototype phase. When real assets replace placeholders:
- Minimum 3 LODs required for any mesh visible beyond 1000 units
- LOD0: Full detail (0–500 units)
- LOD1: 50% poly reduction (500–2000 units)
- LOD2: 75% poly reduction (2000–5000 units)
- Cull: Beyond 5000 units (configurable per asset type)
