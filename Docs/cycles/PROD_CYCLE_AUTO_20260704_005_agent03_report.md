- Hub radius is ~1500u, so 4000u coverage gives full hub + approach paths
- Z scale 10 = 1000u height, covers terrain elevation variation

- Dino actors are currently static mesh placeholders (spheres/cylinders)
- BlockAll ensures they register as solid obstacles for:
  - Player movement (can't walk through)
