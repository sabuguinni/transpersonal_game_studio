```
SkyAtmosphere (SkyAtmosphere_Main)
  - Default settings (UE5 defaults are correct)
  - Label: SkyAtmosphere_Main

SkyLight (SkyLight_Main)
  - Source Type: SLS_CapturedScene
  - Intensity: 1.0
  - Label: SkyLight_Main

ExponentialHeightFog (Fog_Main)
  - Fog Density: 0.02
  - Label: Fog_Main
```

### Spawn Pattern (CORRECT — use this always)

```python
import unreal

dl_cls = unreal.load_class(None, '/Script/Engine.DirectionalLight')
actor = unreal.EditorLevelLibrary.spawn_actor_from_class(
    dl_cls,
    unreal.Vector(0, 0, 10000),
    unreal.Rotator(-45, 0, 0)
)
actor.set_actor_label("Sun_Main")
```

### Anti-Pattern (BROKEN — never use for lighting)

```python
# THIS FAILS for lighting actors:
subsystem = unreal.get_editor_subsystem(unreal.EditorActorSubsystem)
actor = subsystem.spawn_actor_from_class(dl_cls, loc, rot)  # Returns None
```

## Technical Rules — Enforced by Engine Architect

### RULE A — Lighting Actor Spawning
Always use `EditorLevelLibrary.spawn_actor_from_class()` for:
- DirectionalLight, SkyAtmosphere, SkyLight, ExponentialHeightFog
- Any actor that fails with EditorActorSubsystem

### RULE B — Label Format
All actors MUST follow: `Type_Biome_NNN`
- ✅ `Sun_Main`, `TRex_Savana_001`, `Rock_Floresta_042`
- ❌ `FootstepEmitter_CombatZone_QuestTarget_AI_Behavior`

### RULE C — Map Save Protocol
Every session that modifies the map MUST end with:
```python
world = unreal.EditorLevelLibrary.get_editor_world()
unreal.EditorLoadingAndSavingUtils.save_map(world, "/Game/Maps/MinPlayableMap")
```

### RULE D — No C++ in Python-Only Pipeline
The editor runs a pre-compiled binary. `.cpp`/`.h` files written via `github_file_write` are NOT compiled.
All game content MUST be created via `ue5_execute` Python commands.
