## Overview

This document defines the three core survival quests implemented in MinPlayableMap during Cycle 001. All quests are grounded in prehistoric survival realism — no spiritual or mystical content.

## Quest 1: First Hunt

**ID:** QUEST_001_FIRST_HUNT  
**Trigger:** Enter QuestTrigger_FirstHunt zone (coords: 1200, 800, 60)  
**NPC Voice:** Elder_NPC_FirstHunt  
**Audio URL:** https://thdlkizjbpwdndtggleb.supabase.co/storage/v1/object/public/game-assets/tts/1781747242535_Elder_NPC_FirstHunt.mp3

### Objectives
1. Track raptor footprints east of camp (follow particle trail)
2. Approach downwind (stay out of raptor scent cone)
3. Kill 1 raptor with primitive weapon
4. Return meat to camp

### Rewards
- +50 Food resource
- Unlock: Stone Axe crafting recipe
- Unlock: Raptor territory map marker

### Failure Conditions
- Player dies
- Night falls before kill (raptors become pack hunters at night — too dangerous)

### Design Notes
- Raptors at coords ~(1500, 1000) — 3 actors from Agent #12 Combat AI
- Scent cone: player must approach from west (wind direction)
- Footprint particle trail: VFX Agent #17 to implement Niagara trail
- Emotional arc: fear → patience → triumph

## Quest 2: Build Shelter (Campfire)

**ID:** QUEST_002_BUILD_SHELTER  
**Trigger:** Enter QuestTrigger_BuildShelter zone (coords: -500, 700, 55)  
**NPC Voice:** Elder_NPC_BuildShelter  
**Audio URL:** https://thdlkizjbpwdndtggleb.supabase.co/storage/v1/object/public/game-assets/tts/1781747244600_Elder_NPC_BuildShelter.mp3

### Objectives
1. Gather 3 sticks (Resource_Stick_001/002/003 actors in world)
2. Open crafting menu (press C near CraftingStation_Base)
3. Craft Campfire from 3 sticks
4. Place campfire at designated spot

### Crafting Recipe: Campfire
```
Input:  3x Stick
Output: 1x Campfire (placeable actor)
Effect: Illuminates 800 unit radius, repels predators at night, restores warmth stat
```

### Rewards
- +30 Warmth stat
- Predators avoid 800-unit radius at night
- Unlock: Water Container crafting recipe

### Failure Conditions
- Night falls without campfire (cold damage begins, predators attack camp)

### Design Notes
- Resource_Stick actors placed at (-400,800), (-200,900), (300,700), (700,-400)
- CraftingStation_Base at (0, 300) — visual marker for crafting UI
- Emotional arc: urgency → resourcefulness → safety

## Quest 3: Water Source

**ID:** QUEST_003_WATER_SOURCE  
**Trigger:** Enter QuestTrigger_WaterSource zone (coords: 200, -1200, 55)  
**NPC Voice:** (to be generated by Agent #16 Audio)

### Objectives
1. Find the river (explore south of camp)
2. Gather 1 rock + 1 leaf (Resource_Rock + Resource_Leaf actors)
3. Craft Water Container at CraftingStation_Base
4. Fill container at river (interact with water volume)
5. Return to camp with water

### Crafting Recipe: Water Container
```
Input:  1x Rock + 1x Leaf
Output: 1x Water Container (inventory item)
Effect: Carries 3 uses of water, restores Thirst stat by 40 per use
```

### Rewards
- +40 Thirst stat (immediate use)
- Unlock: River map marker
- Unlock: Advanced crafting tier

### Failure Conditions
- Thirst reaches 0 (player collapses)
- Predator encounter at river (Brachiosaurus territory)

### Design Notes
- River area at ~(200, -1200) — Agent #05 World Generator terrain
- Brachiosaurus at (0, -1500) — peaceful unless provoked
- Emotional arc: desperation → discovery → relief

## Crafting System — Recipe Definitions

### Stone Axe
```
Input:  2x Rock + 1x Stick
Output: 1x Stone Axe (equippable weapon)
Damage: 25 per hit vs dinosaurs
Durability: 15 hits
```

### Campfire
```
Input:  3x Stick
Output: 1x Campfire (placeable)
Effect: Light radius 800u, warmth +30, predator repellent
```

### Water Container
```
Input:  1x Rock + 1x Leaf
Output: 1x Water Container (inventory)
Capacity: 3 uses × 40 Thirst restore
```
