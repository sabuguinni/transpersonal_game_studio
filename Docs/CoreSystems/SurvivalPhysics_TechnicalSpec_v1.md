## 3. Survival Stats System

### Stats Architecture
All stats tick every 1 second (server-authoritative):

```
Health:   100 → 0 (death)    | Drain: combat damage, fall damage, starvation
Hunger:   100 → 0 (starving) | Drain: 0.5/min baseline, 2.0/min sprinting
Thirst:   100 → 0 (dehydrate)| Drain: 0.8/min baseline, 3.0/min in heat
Stamina:  100 → 0 (exhausted)| Drain: sprinting, climbing; Regen: resting
Fear:     0 → 100 (panic)    | Increase: dino proximity; Decrease: safe zones
```

### Survival Anchor
- Label: `SurvivalSystem_Anchor`
- Location: 200 units above PlayerStart
- Visual: Cyan PointLight (intensity 300, radius 400)
- Purpose: Debug marker confirming survival system is active

## 4. Terrain Collision Configuration

### Ground Mesh Collision
- Collision Profile: `BlockAll`
- Physics Material: `PM_Terrain_Default`
- Walking Surface Normal Tolerance: 45° max slope

### Ragdoll Physics (Dinosaurs)
- Enabled on death trigger
- Mass: TRex=8000kg, Raptor=80kg, Brach=50000kg
- Linear Damping: 0.1
- Angular Damping: 0.05
