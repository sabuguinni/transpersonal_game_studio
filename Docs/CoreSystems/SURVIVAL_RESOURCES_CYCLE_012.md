## Architecture Notes

### Survival System Design (Python-driven)
Since C++ compilation is inert in the current editor binary, all survival mechanics
are implemented via UE5 Python + Blueprint-compatible actor placement:

- **Flint rocks** → Player walks near → Blueprint overlap event → add to inventory
- **Berry bushes** → Player walks near → Blueprint overlap event → restore hunger
- **Patrol waypoints** → Dino AI Blueprint reads these labels → follows circuit

### Resource Interaction Pattern
```
Player (TranspersonalCharacter)
  └── Sphere Overlap Trigger (Blueprint)
        ├── Flint_Rock_NNN → AddItem("Flint", 1-3)
        └── BerryBush_NNN → AddHunger(+15), Destroy(actor)
```

### Patrol Circuit Pattern
```
DinoActor (Blueprint)
  └── OnBeginPlay → FindActorsWithTag("PatrolWaypoint")
        └── MoveToLocation(waypoints[currentIndex])
              └── OnReached → currentIndex = (currentIndex+1) % 4
```
