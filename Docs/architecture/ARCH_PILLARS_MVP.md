## Architecture Decisions (Engine Architect #02)

### Why Python-only (no C++ compilation)
The UE5 editor runs a pre-compiled binary. `.cpp`/`.h` files written via `github_file_write` are **never compiled** during production cycles. All game content is created via `ue5_execute` Python commands that run directly in the editor via Remote Control API.

### Why 8 pillars
These 8 systems represent the minimum for a playable scene:
- Without P1/P2/P3: scene is black, unplayable
- Without P4: no terrain, character falls through void
- Without P5: no spawn point, game cannot start
- Without P6: no dinosaurs, no gameplay threat
- Without P7: empty world, no visual interest
- Without P8: AI cannot pathfind, dinosaurs are static

### Lighting configuration (Cretaceous atmosphere)
```python
# Sun — warm golden Cretaceous light
intensity = 8.0
light_color = LinearColor(1.0, 0.9, 0.7, 1.0)  # warm amber
pitch = -50  # mid-afternoon angle
atmosphere_sun_light = True

# SkyLight — real-time capture for dynamic reflections
intensity = 1.5
real_time_capture = True

# HeightFog — subtle Cretaceous haze
fog_density = 0.02
```
