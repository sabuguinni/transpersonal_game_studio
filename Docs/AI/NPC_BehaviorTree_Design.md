## Behavior Tree State Machine

```
[ROOT]
  └─ [Selector: Survival Priority]
       ├─ [Sequence: FLEE] ← Highest priority
       │    ├─ Condition: IsDangerNearby (radius 2000u)
       │    └─ Task: FleeToSafeZone
       │
       ├─ [Sequence: ALERT]
       │    ├─ Condition: IsAlertSignalReceived
       │    └─ Task: BroadcastAlert + SeekCover
       │
       ├─ [Sequence: COMBAT_SUPPORT]
       │    ├─ Condition: IsPlayerInCombat AND IsAlly
       │    └─ Task: ThrowRock OR CreateDistraction
       │
       ├─ [Sequence: DAILY_ROUTINE]
       │    ├─ Condition: IsTimeOfDay(Morning)
       │    └─ Task: PatrolRoute → GatherResources → ReturnCamp
       │
       └─ [Sequence: IDLE]
            └─ Task: IdleAnimation + OccasionalDialogue
```

## NPC Memory System

NPCs remember:
- **Last seen player location** (fades after 5 minutes real-time)
- **Last dinosaur sighting** (position + species + time)
- **Danger events** (ground shake, roar, blood trail)
- **Resource locations** (berry bush, water source, prey trail)

Memory is stored per-NPC as a simple array of FNPCMemoryEntry structs.
Max 16 entries per NPC — oldest entry evicted when full (FIFO).
