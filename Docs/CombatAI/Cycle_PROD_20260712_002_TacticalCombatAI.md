## Anti-duplication audit (per Brain rule `hugo_combat_label_consistency_v1`)
**Important finding:** the Remote Control bridge's `result` field only ever returns
`{"ReturnValue": true}` regardless of what the Python script computes or prints —
it confirms execution success but does not surface stdout/return values back to
this session. This has been true across all calls this cycle (audit query, verification
query). Because of this, I could not visually confirm the exact prior counts of
`CombatZone_*` / `BehaviorTag_*` labels reported in earlier cycles (e.g. the ~191
`CombatZone_Raptor` duplicates flagged in Brain memory).

**Mitigation used:** rather than relying on log visibility, I embedded the dedup
logic directly inside the idempotent script itself:
- For each dinosaur type (TRex, Raptor, Trike, Brachio) found near the hub (2100,2400,
  radius 3500), the script searches for an EXISTING `CombatZone_<Type>` and
  `BehaviorTag_<Type>` actor first.
- If found → reused (relabeled in place, no new actor spawned).
- If NOT found → exactly one new `TextRenderActor` created and labeled.
- This guarantees at most one CombatZone/BehaviorTag pair per dinosaur type is
  ever added in this cycle, regardless of what pre-existing duplicate count was
  present, and does not add to the duplicate pile described in the Brain memory.

This does NOT retroactively clean up the ~191+185 pre-existing duplicates reported
in Brain memory — that requires a batch-delete pass (recommended as next step,
see below) which was out of scope for a single-cycle tactical AI pass.

## Tactical combat AI parameters applied (as Actor Tags on existing dinosaurs)
Building on NPC Behavior Agent #11's `Behavior_*` tags from this same cycle, applied
combat-specific tactical parameters directly onto the existing tagged dinosaur actors
(no new dinosaurs spawned — reused actors placed by Animation Agent #10):

- **T-Rex** — `AI_Combat=ApexAmbush;Aggro=800;AttackWindup=1.2s;RetreatHP=0.15;CommentOnFlank=true`
  Design intent: apex predator commits to an obvious wind-up before the killing blow,
  giving the player a readable dodge window (Naughty Dog "fair difficulty" principle).
  Retreats at 15% HP rather than fighting to the death — a wounded T-Rex disengaging
  reads as a believable animal, not a health-bar monster.
- **Raptor (pack)** — `AI_Combat=PackFlank;Aggro=600;SyncAttack=true;CoordSignal=Screech;RetreatHP=0.3`
  Design intent: raptors coordinate a flank via an audible screech cue (telegraphed,
  not hidden) so an alert player can reposition against a pincer. One raptor breaks
  off to flee at 30% HP, thinning the pack mid-fight for pacing.
- **Triceratops** — `AI_Combat=DefensiveCharge;Aggro=400;ChargeThreshold=0.5;Cooldown=4s`
  Design intent: territorial defense, not predation — charges only when provoked/cornered,
  matching herbivore ecology (no unprovoked hunting behavior).
- **Brachiosaurus** — `AI_Combat=Passive;NoAggro=true;FleeOnly=true`
  Design intent: sauropods never initiate combat; confirmed consistent with #11's
  `Behavior_PassiveGrazer` tag from this same cycle.
