# Quest Dialogue System - Visual Design Concept

## Interface Design Vision

### Stone Age UI Aesthetic
- **Primary Materials**: Weathered stone, carved bone, animal hide textures
- **Color Palette**: Earth tones - ochre, burnt sienna, charcoal black, bone white
- **Typography**: Hand-carved appearance with primitive symbols and runic-style text

### Dialogue Interface Layout

#### Main Dialogue Box
- **Background**: Flat stone tablet with rough, weathered edges
- **Border**: Carved primitive symbols representing survival elements (spears, fire, animals)
- **Text Area**: Etched appearance as if carved into stone
- **NPC Portrait**: Circular frame made of woven plant fibers or animal hide

#### Response Options
- **Format**: Separate stone fragments or bone pieces for each response
- **Interaction**: Glow effect using warm firelight when hovered
- **Selection**: Carved arrow or primitive pointing symbol
- **Maximum**: 4 response options to maintain visual balance

#### Visual Indicators
- **Quest Start**: Glowing ember or small flame icon
- **Quest Progress**: Stone markers or notched counting sticks
- **Important Information**: Highlighted with ochre paint or blood-red staining

### Environmental Integration
- **Backdrop**: Primitive camp setting with wooden spears, stone tools, campfire
- **Lighting**: Warm firelight creating dramatic shadows
- **Atmosphere**: Dust particles, smoke wisps, natural weathering effects

### NPC Character Design
- **Appearance**: Weathered human survivors in animal hide clothing
- **Expression**: Urgent, concerned, battle-hardened
- **Accessories**: Primitive jewelry, scars, tribal markings
- **Posture**: Alert, ready for danger, survival-focused

### Dialogue Content Themes
- **Survival Advice**: Hunting techniques, shelter building, resource gathering
- **Danger Warnings**: Dinosaur behavior, territorial boundaries, pack hunting patterns
- **Resource Information**: Water sources, tool materials, safe zones
- **Quest Objectives**: Hunting targets, exploration goals, survival challenges

### Technical Implementation Notes
- **Texture Resolution**: High-detail stone and hide textures for close-up UI elements
- **Animation**: Subtle particle effects for dust and ember glow
- **Sound Design**: Stone scraping, leather creaking, fire crackling ambient sounds
- **Accessibility**: High contrast between text and background for readability

This concept art would show a dialogue interface that feels authentic to the prehistoric survival setting while maintaining modern usability standards.
