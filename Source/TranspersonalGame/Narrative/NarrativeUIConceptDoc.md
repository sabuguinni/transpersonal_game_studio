# Narrative UI System Concept Design

## Visual Design Philosophy
The narrative interface should feel like ancient stone tablets and cave paintings, immersing players in the prehistoric world without breaking immersion with modern UI elements.

## Core Visual Elements

### Stone Tablet Interface
- **Material**: Weathered limestone texture with carved indentations
- **Lighting**: Warm amber glow from torchlight, casting dramatic shadows
- **Typography**: Primitive pictographs and simple carved symbols
- **Color Palette**: Earth tones - ochre, burnt sienna, charcoal black

### Dialogue System
- **Speech Bubbles**: Carved bone fragments with etched text
- **Character Portraits**: Stone relief carvings showing tribal faces
- **Voice Indicators**: Glowing ember effects when characters speak
- **Response Options**: Multiple carved symbols representing different emotional responses

### Quest Tracking
- **Progress Markers**: Stone cairns and primitive tally marks
- **Map Integration**: Cave painting style world map with hunting grounds marked
- **Objective Icons**: Tribal pictographs (spear for hunt, fire for gather, footprints for track)

### Narrative Moments
- **Cutscene Frames**: Framed like cave painting sequences
- **Story Reveals**: Ancient scroll unfurling animation
- **Memory Flashbacks**: Smoke and ember particle effects transitioning to sepia tones

## Technical Implementation Notes
- Use UMG widgets with custom materials for stone/bone textures
- Implement particle systems for torch lighting effects
- Create custom fonts based on primitive symbol sets
- Utilize Niagara for atmospheric smoke and ember effects

## Audio Integration
- Stone scraping sounds for UI navigation
- Tribal drum beats for important narrative moments
- Wind and fire ambience for background atmosphere
- Character voice lines with reverb suggesting cave acoustics
