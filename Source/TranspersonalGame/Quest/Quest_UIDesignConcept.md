# Quest System UI Design Concept

## Visual Theme: Prehistoric Stone Tablet Interface

### Core Design Elements
- **Base Material**: Weathered stone texture with carved symbols
- **Accent Color**: Glowing amber/orange for active elements
- **Typography**: Primitive cave painting style icons and symbols
- **Lighting**: Warm firelight illumination from below

### Quest Display Components

#### Main Quest Panel
- Large stone tablet background (1024x768px)
- Quest title carved at top in runic-style font
- Objective list with primitive checkbox symbols (spear tips, claw marks)
- Progress bars using amber crystal fill animation

#### Quest Types & Icons
1. **Hunt Quests**: Dinosaur skull icon with target count
2. **Gather Quests**: Resource bundle icon (sticks, stones, crystals)
3. **Explore Quests**: Cave entrance or mountain peak symbol
4. **Craft Quests**: Primitive tool crossed with materials
5. **Survival Quests**: Campfire or shelter icon

#### Interactive Elements
- **Accept Button**: Carved "YES" with handprint symbol
- **Decline Button**: Carved "NO" with crossed spears
- **Track Button**: Arrow pointing forward with footprint trail
- **Abandon Button**: Broken chain link symbol

### Animation States
- **Quest Available**: Gentle amber glow pulse
- **Quest Active**: Steady amber illumination
- **Quest Complete**: Bright golden flash then fade to completion mark
- **Quest Failed**: Red ember glow then fade to gray stone

### Audio Cues
- Quest accepted: Stone grinding sound + tribal drum beat
- Quest completed: Crystal chime + satisfied grunt
- Quest failed: Stone cracking sound + disappointed sigh

This design maintains immersion in the prehistoric setting while providing clear, functional quest management for survival gameplay.
